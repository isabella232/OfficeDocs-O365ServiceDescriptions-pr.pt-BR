# <a name="insights"></a>Informações

O SharePoint Online é uma coleção de ferramentas e tecnologias baseadas na Web que ajudam sua organização a armazenar, compartilhar e gerenciar informações digitais. Criado com base no Microsoft SharePoint Server 2013, este serviço hospedado é ideal para trabalhar com projetos, armazenar dados e documentos em um local central e compartilhar informações. Os Recursos de Ideias de pesquisas a seguir permitem que as organizações usem as informações presentes em bancos de dados, relatórios e aplicativos empresariais para atender a necessidades específicas de seus negócios.
  
## <a name="business-intelligence-center"></a>Central de Business Intelligence
<a name="bkmk_BusinessIntelligenceCenter"> </a>

O site da Central de Business Intelligence permite que você armazene conexões de dados, pastas de trabalho, relatórios, scorecards e painéis em um local seguro central. Saiba mais sobre a [configuração de uma Central de Business Intelligence](https://go.microsoft.com/fwlink/p/?LinkId=270883).
  
## <a name="calculated-measures-and-members"></a>Calculated Measures and Members
<a name="bkmk_CalculatedMeasuresandMembers"> </a>

Quando você cria relatórios de tabela e gráfico dinâmicos que usam dados armazenados nos Serviços de Análise do SQL Server, é possível criar medidas e membros calculados para esses relatórios. Saiba mais sobre [recursos de business intelligence nos Serviços do Excel](https://go.microsoft.com/fwlink/p/?LinkId=271025).
  
## <a name="data-connection-library"></a>Data Connection Library
<a name="bkmk_DataConnectionLibrary"> </a>

Uma biblioteca de conexões de dados é uma biblioteca especial do SharePoint que foi projetada para armazenar e gerenciar conexões de dados compartilhados. Elas podem incluir arquivos ODC que você cria por meio do Excel. Saiba mais sobre [Bibliotecas de conexões de dados confiáveis no SharePoint Server 2013](https://go.microsoft.com/fwlink/p/?LinkId=271026).
  
## <a name="decoupled-pivottables-and-pivotcharts"></a>Decoupled PivotTables and PivotCharts
<a name="bkmk_DecoupledPivotTablesPivotCharts"> </a>

Quando você criar um relatório de Gráfico Dinâmico, será criado automaticamente um relatório de Tabela Dinâmica correspondente. Agora, no Excel 2013, você não precisa exibir um relatório de tabela dinâmica na mesma planilha do relatório correspondente de gráfico dinâmico. Você pode exibir apenas o relatório de gráfico dinâmico. Esse recurso é chamado de "tabelas e gráficos dinâmicos separados". Saiba mais sobre [tabelas e gráficos dinâmicos](https://go.microsoft.com/fwlink/?LinkId=271027).
  
## <a name="excel-services"></a>Serviços do Excel
<a name="bkmk_ExcelServices"> </a>

Os Serviços do Excel permitem compartilhar pastas de trabalho com outras pessoas, interagir com relatórios, visualizar e explorar informações em uma janela de navegador e atualizar dados. Os Serviços do Excel também permitem que você compartilhe pastas de trabalho inteiras ou apenas parte delas e controle as informações que são exibidas nelas. No SharePoint 2016, os recursos dos Serviços do Excel estão disponíveis no Excel Online no Servidor do Office Online. Os Serviços do Excel requerem o Servidor do Office Online no ambiente local. Saiba mais sobre os [Serviços do Excel](https://go.microsoft.com/fwlink/p/?LinkId=271028).
  
## <a name="excel-web-access-web-part-for-sharepoint"></a>Web Part do Excel Web Access para SharePoint
<a name="bkmk_WebPart"> </a>

Você pode usar a Web Part do Excel Web Access para exibir conteúdo em Excel em um site do SharePoint.
  
## <a name="field-list-and-field-support"></a>Lista de campos e suporte a campos
<a name="bkmk_FieldlListFieldSupport"> </a>

O Excel Online permite que você abra a Lista de campos e Fonte de campos para relatórios de Gráfico Dinâmico e relatórios de Tabela Dinâmica que são exibidos nas janelas do navegador. Esse recurso facilita a alteração temporária das informações exibidas em um relatório de tabela dinâmica ou de gráfico dinâmico sem precisar abrir o Excel. Saiba mais sobre [Business Intelligence no Excel e nos Serviços do Excel](https://go.microsoft.com/fwlink/p/?LinkId=271029).
  
## <a name="filter-enhancements"></a>Filter Enhancements
<a name="bkmk_FilterEnhancements"> </a>

É mais fácil criar segmentações de dados e vinculá-las a relatórios. Você pode ajustar o tamanho e a orientação de uma segmentação de dados. Você também pode formatar segmentações de dados aplicando estilos internos ou definindo seu próprio estilo.
  
## <a name="filter-search"></a>Pesquisa de filtro
<a name="bkmk_FilterSearch"> </a>

Você pode pesquisar os resultados de filtro, o que facilita a navegação para um item específico de uma lista extensa. Saiba mais sobre o [uso de filtros e segmentações de dados](https://go.microsoft.com/fwlink/p/?LinkId=271030).
  
## <a name="odc-file-support"></a>Suporte para arquivo ODC
<a name="bkmk_ODCFileSupport"> </a>

Os arquivos ODC permitem gerenciar, compartilhar e reutilizar conexões de dados com fontes de dados do Enterprise. As conexões de dados a determinadas fontes de dados podem ser configuradas e gerenciadas centralmente pelo pessoal de TI e ficar disponível para qualquer pasta de trabalho que consome arquivos ODC. Você pode manter conjuntos de arquivos ODC em locais central no farm do SharePoint e você pode editar pastas de trabalho do Excel para utilizar esses arquivos.
  
## <a name="performancepoint-services"></a>Serviços do PerformancePoint
<a name="bkmk_PerformancePointServices"> </a>

Indisponível para clientes do SharePoint Online. Os Serviços do PerformancePoint permitem que os clientes do SharePoint Server 2013 criem painéis interativos que exibam KPIs (indicadores de desempenho principais) e virtualizações de dados sob a forma de scorecards, relatórios e filtros. Saiba mais sobre os [Serviços do PerformancePoint](https://go.microsoft.com/fwlink/p/?LinkId=271032).
  
## <a name="power-bi-web-part"></a>Web Part do Power BI
<a name="bkmk_PowerBI"> </a>

O Power BI é um pacote de ferramentas de análise de negócios baseado na nuvem que permite que qualquer pessoa se conecte, visualize e analise dados com mais rapidez, eficiência e compreensão. Ele conecta os usuários a uma ampla variedade de dados ao vivo por meio de painéis simples, fornece relatórios interativos e visualizações atraentes para dar vida aos dados. Saiba mais sobre o Power BI em [Introdução ao serviço Power BI](https://go.microsoft.com/fwlink/?linkid=844671).
  
## <a name="power-pivot-for-excel-in-sharepoint"></a>Power Pivot para Excel no SharePoint
<a name="bkmk_PowerPivot"> </a>

Com o Power Pivot para Excel, você pode realizar análises de dados avançadas e criar modelos de dados sofisticados. Você também pode combinar grandes volumes de dados a partir de várias fontes, analisar dados rapidamente e compartilhar as ideias de pesquisa. Com o SharePoint Online, você pode usar Excel Online para visualizar e interagir com quadros, tabelas, relatórios do PivotChart e relatórios do PivotTable em uma janela do navegador. Você também pode usar a Web Part do Excel Web Access para exibir conteúdo em Excel em um site do SharePoint. As folhas do Power Pivot devem ser criadas em Excel antes de serem visualizadas no SharePoint. Saiba mais sobre o uso do [Power Pivot em Excel](https://go.microsoft.com/fwlink/p/?LinkId=271036).
  
## <a name="power-view-for-excel-in-sharepoint"></a>Power View for Excel in SharePoint
<a name="bkmk_PowerView"> </a>

 Com o Power View para Excel, você pode visualizar e interagir com dados modelados usando visualizações, animações e consultas inteligentes altamente interativas. Com SharePoint Online, você usa Excel Online para visualizar e interagir com pastas de trabalho que podem conter folhas do Power View em uma janela do navegador. Você também pode usar a Web Part do Excel Web Access para exibir conteúdo em Excel em um site do SharePoint. As folhas do Power View devem ser criadas em Excel antes de serem visualizadas no SharePoint. Saiba mais sobre o uso do [Power View em Excel](https://go.microsoft.com/fwlink/p/?LinkId=271035).
  
## <a name="quick-explore"></a>Exploração Rápida
<a name="bkmk_QuickExplore"> </a>

A Exploração Rápida permite que as organizações selecionem um valor em um relatório de gráfico dinâmico ou tabela dinâmica e visualizem informações adicionais sobre esse valor. Por exemplo, se você está visualizando um relatório de vendas e seleciona um valor que representa os valores totais de venda, é possível usar a Exploração Rápida para exibir informações adicionais como valores de venda em diferentes categorias de produto e áreas geográficas. Saiba mais sobre a [Exploração rápida](https://go.microsoft.com/fwlink/p/?LinkId=271037).
  
## <a name="refresh-excel-workbooks-interactively"></a>Atualizar Pastas de Trabalho do Excel Interativamente
<a name="bkmk_RefreshExcel"> </a>

Permite ver dados atualizados em suas pastas de trabalho do Excel Online que estão conectados a fontes de dados externos (por exemplo, bancos de dados SQL do MS). Observe que você deve conectar suas pastas de trabalho a fontes de dados externas no Excel antes de exibi-las no Excel Online.
  
## <a name="sql-server-reporting-services-ssrs-integrated-mode"></a>SQL Server Reporting Services (SSRS) Integrated Mode
<a name="bkmk_SQLServerReportingServicesIntegratedMode"> </a>

Quando os Serviços de Relatório do SQL Server 2012 são integrados ao SharePoint, você pode usar colaboração avançada do SharePoint e Recursos centralizados de gerenciamento de documentos para o conteúdo dos Serviços de Relatório do SQL Server 2012. Os relatórios são armazenados nas bibliotecas de documentos do SharePoint junto com outros relatórios, como arquivos do Excel Online. As permissões do SharePoint podem ser usadas para controlar o acesso ao conteúdo dos Serviços de Relatório, e as pessoas podem iniciar o Designer de Relatórios, Designer de Modelos e Construtor de Relatórios em uma biblioteca de documentos. Saiba mais sobre o [Servidor de Relatórios do Reporting Services](https://go.microsoft.com/fwlink/p/?LinkId=271039).
  
## <a name="sql-server-reporting-services-ssrs-web-part"></a>Web Part do SQL Reporting Services
<a name="bkmk_SSRSWebPart"> </a>

A Web Part dos Serviços de Relatório do SQL Server 2012 permite que as pessoas exibam um relatório existente de Serviços de relatório na própria Web Part em um site do SharePoint. Se o relatório de Serviços de Relatório contém parâmetros, as pessoas podem vincular filtros a esse relatório usando as configurações de Web Part. Saiba mais sobre a [Web Part Visualizador de relatórios](https://go.microsoft.com/fwlink/p/?LinkId=271041).
  
## <a name="timeline-slicer"></a>Timeline Slicer
<a name="bkmk_TimelineSlicer"> </a>

Uma segmentação de dados da linho do tempo é um controle que pode ser adicionado a um painel criado por meio do Excel. Um controle da linha do tempo permite que as pessoas exibam informações de um período específico. Saiba mais sobre [Business intelligence nos Serviços do Excel](https://go.microsoft.com/fwlink/p/?LinkId=271042).
  
## <a name="use-and-interact-with-excel-workbooks-containing-data-models"></a>Usar e Interagir com Pastas de Trabalho do Excel que Contêm Modelos de Dados
<a name="bkmk_DataModels"> </a>

As Pastas de Trabalho do Excel às vezes contêm modelos de dados do PowerPivot que permitem executar poderosas análises de dados, lidar com volumes grandes de dados de várias origens, analisar dados rapidamente e compartilhar informações.
  
## <a name="user-defined-functions"></a>Funções Definidas pelo Usuário
<a name="bkmk_UserDefinedFunctions"> </a>

As funções definidas pelo usuário (UDFs) oferecem a capacidade de usar fórmulas em células de planilha do Excel Online para chamar funções personalizadas gravadas em código gerenciado. As UDFs são um recurso de programação avançada que permite que sua empresa desenvolva e conecte código personalizado em planilhas do Excel Online. Você pode criar UDFs para chamar funções matemáticas personalizadas, obter dados de fontes de dados personalizadas para planilhas e chamar serviços Web a partir das UDFs.
  
## <a name="visio-services"></a>Serviços do Visio
<a name="bkmk_VisioServices"> </a>

O Visio Services é um aplicativo de serviço que permite compartilhar e exibir os arquivos de desenho do Microsoft Visio Drawing (\*.vsdx) e do Visio 2010 Web Drawing (\*.vdw). O serviços do Visio não está mais disponível com o SharePoint Online. Os serviços do Visio continuarão disponíveis no SharePoint Server 2010, no SharePoint Server 2013 e no SharePoint Server 2016. Saiba mais sobre os [Serviços do Visio](https://go.microsoft.com/fwlink/p/?LinkID=271043).

<a name="bkmk_VisioServices"> </a>

Para exibir a disponibilidade de recursos nos planos do Office 365 nas opções autônomas e nas soluções locais, consulte [Descrição do serviço SharePoint Online](sharepoint-online-service-description.md).
